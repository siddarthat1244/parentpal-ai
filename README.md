# 👶 ParentPal AI

ParentPal AI is an AI-powered parenting assistant built using React, TypeScript, Node.js, Express, and AI.

---

## Features

- 🤖 AI Parenting Assistant
- 👶 Child Profiles
- 💬 Personalized AI Chat
- 💾 Chat History
- 🎨 Material UI
- ⚡ Fast Node.js Backend
- 🧠 AI Provider Pattern

---

## Tech Stack

### Frontend

- React
- TypeScript
- Material UI
- Axios
- React Router

### Backend

- Node.js
- Express
- TypeScript

### AI

- Groq
- OpenAI (future)

---

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run dev
```
